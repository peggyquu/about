<br>
在設計的領域中，一邊工作一邊學習，不斷地獲取新資訊 成為生活與設計的養分也是必備的態度之一，有感於網路 與行動裝置的使用頻繁，在品辰工作近半年左右，開始網 頁設計切版的課程進修，或是購買網路課程，如 hahow、 udemy 等平台在家自學設計相關課程。<br>
